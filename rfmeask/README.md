## Программа управления позиционером.

Реализация управления строится на иерархиях классов, корнями которых являются Device и LogicalDevice.

Класс Device отвечает за управление и контроль работоспособности физических устройств, подключаемых к компьютеру управления.

Класс LogicalDevice комплексирует информацию от разных устройств, и является предком интерфейсов для логических абстракций, удобных для управления изделием (например, ось). LogicalDevice так же получает получает информацию о работоспособности входящих в него устройств, тем самым узнавая собственную работоспособность (с помощью класса DeviceChecker). 

Объекты LogicalDevice управляют объектами Device непосредственно, через вызовы функций. Для передачи обратной информации (от Device к LogicalDevice) между объектами использована система сигналов (класс signal).

Наследники класса Device реализуют управление зависимыми от них устройствами через объекты-комуникаторы, реализованные в виде синглтонов. Данные объекты отвечают за потокобезопастность коммуникации с устройствами. 

Внешнее управление реализуется с помощью интерфейсов управления. (Консоль, протоколы управления). Также отдельно выстроена система уведомлений (классы Notifier и NotifyTheme). Уведомления используются для передачи информации о событиях на стороне аппаратуры (аппаратный сбой, выход в заданную позицию).

Предполагается, что программа управления может быть расширенна на любой тип сервоусилителя и работать с разными типами измерительно-синхронизующего оборудования через приведение к классам Device и LogicalDevice.


## Инструкция по установке:

0. Скачать с сабмодулями или обновить сабмодули, если проект уже скачан.
```sh
git clone https://radioline.visualstudio.com/rfmeask/_git/rfmeask
```

1. Установить систему сборки licant. 
```sh
python3 -m pip install licant 
```

2. Перейти в директорий проекта, Собрать проект:
```sh
cd rfmeask
python3 ./make.py
```

3. Если сборка завершилась удачно, запускать файл 
```sh
rfmeas
```
Конфигурация системы в файле
```sh
settings.json
```
